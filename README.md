# React Practice Projects ⚛️

This repository contains small React projects built while learning React concepts such as hooks, routing, and API integration.

---

## 📂 Projects

### 💱 Currency Converter

**Folder:** `06CurrencyConverter`

A React application that converts currencies using live exchange rates.

**Concepts Used**

* React Hooks (`useState`, `useEffect`)
* Custom Hooks
* API fetching
* Controlled inputs
* Component reuse

**Features**

* Select currencies
* Enter amount
* Convert between currencies
* Swap currencies

---

### 🌐 React Router Project

**Folder:** `07reactRouter`

A project demonstrating navigation using **React Router**.

**Concepts Used**

* `createBrowserRouter`
* `RouterProvider`
* `Route`
* `Outlet`
* `useParams`

**Pages**

* Home
* About
* Contact

## 🚀 Running the Project

1. Clone the repository

```
git clone https://github.com/your-username/react-projects.git
```

2. Navigate to the project

```
cd 06CurrencyConverter
```

or

```
cd 07reactRouter
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm run dev
```
